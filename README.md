A professional, interactive calculator built with HTML, CSS, and JavaScript.  
Supports three modes: Standard, BMI, and Loan Calculator.  

This project demonstrates modern UI design, creative styling, and multi-mode calculator functionality.  

Features

- Standard Calculator: Basic arithmetic with clear, backspace, percent, and square root functions.  
- BMI Calculator: Calculates Body Mass Index from weight (kg) and height (cm).  
- Loan Calculator: Computes monthly payment based on principal, annual interest rate, and term.  
- Responsive Design: Works on desktop and mobile.  
- Interactive & Eye-Catching UI: Smooth animations, hover effects, and gradients for professional appearance.
